OnPickupStreamIn

注意

这个回调是在omp v1.1.0.2612中添加的，在以前的版本中不起作用！

描述​

当拾取物进入玩家的可视范围时触发该回调。

参数说明
pickupid拾取物ID（由CreatePickup函数返回）
playerid触发流加载的玩家ID

返回值​

该回调在游戏模式中总是优先触发。

示例​

new g_PickupHealth;

public OnGameModeInit()
{
    g_PickupHealth = CreatePickup(1240, 2, 2009.8474, 1218.0459, 10.8175);
    return 1;
}

public OnPickupStreamIn(pickupid, playerid)
{
    if (pickupid == g_PickupHealth)
    {
        printf("生命值拾取物已为玩家 %d 完成流加载", playerid);
    }
    return 1;
}

相关回调​

以下回调可能与当前回调存在关联：

  • OnPlayerPickUpPickup：当玩家拾取物品时触发
  • OnPickupStreamOut：当拾取物离开玩家可视范围时触发

相关函数​

以下函数可能与当前回调相关：

  • CreatePickup：创建拾取物
  • DestroyPickup：销毁拾取物
