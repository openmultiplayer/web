Ressources

Outils​

  • Community Compiler - Une version largement mise à jour du compilateur, avec de nombreux correctifs et améliorations.
  • sampctl - Gestionnaire de packages pour installer des bibliothèques et faire fonctionner votre serveur.
  • Plugin Runner - Outil pour exécuter une version allégée du serveur directement depuis la ligne de commande (aucun server.cfg requis) pour les tests plugins.
  • Plugin Boilerplate - Faciliter un plugin avec le passe-partout.
  • SA:MP Plugin Template Library - Créez vos propres plugins très facilement et rapidement avec cette bibliothèque de modèles.
  • SA-MP Fiddle - Un endroit pour tester vos scripts (extraits, PoC, débogage, etc.) et les partager avec d'autres.
  • Pawn Syntax - Sublime - Pawn se complète automatiquement pour Sublime Text.
  • Pawn Syntax - Visual Marketplace - Pawn se complète automatiquement pour Visual Studio Code.
  • Éditeur de zone SA-MP - Éditeur de zone pour créer Area et tout.
  • Éditeur de carte SA-MP - Éditeur de carte populaire pour SA:MP.

Bibliothèques​

  • samp-stdlib - Les versions mises à jour de la valeur par défaut incluent. const-correct, documenté et complet.
  • fixes.inc - Corrections extrêmement optimisées pour un grand nombre de bugs du serveur SA:MP. Brancher et utiliser.
  • YSI-Includes - La bibliothèque la plus ancienne, la plus grande, la plus testée et la mieux prise en charge pour SA:MP/pawn, fournissant un énorme nombre de nouvelles fonctionnalités de jeu et de langue.
  • foreach - foreach standalone include (version non y_iterate).
  • amx_assembly - Accès de bas niveau au script de pion lui-même.
  • md-sort - Trier des tableaux multidimensionnels dans Pawn. Modifie les pointeurs internes du tableau au lieu de copier autour des données.
  • indirection - Système pour passer des pointeurs et appeler des fonctions indirectement, avec des méthodes abstraites pour personnaliser les appels.
  • code-parse.inc - Analysez et personnalisez le code de pion au moment de la compilation.
  • Informations étendues sur les véhicules - De grandes quantités de drapeaux de véhicules d'un seul joueur dans une base de données SQLite.
  • sqlitei - API de haut niveau pour les fonctions SA-MP SQLite.
  • weapon-config - Système de dégâts plus cohérent et réactif avec de nombreuses nouvelles fonctionnalités.
  • samp-geoip - Informations géographiques basées sur l'adresse IP.
  • progress2 - Affichez l'heure, la santé, le carburant du véhicule et d'autres éléments sous forme de barre progressive.
  • weapon-data - Données d'armes finement réglées et personnalisables.
  • MV_Youtube - API Youtube Converter pour le flux audio.
  • ** [Instructions préparées MySQL] (https://github.com/PatrickGTR/MySQL-Prepared-Statements) ** - Émulation d'instructions préparées pour le plug-in PAWN MySQL.
  • samp-server-weapons - Armes totalement contrôlées par le serveur.
  • actor_robbery - Inspiré du vol de magasin GTA V. actor_robbery.inc l'imite !
  • samp-aviation - Un pilote automatique de base basé sur l'altitude et le roulis pour SA-MP.
  • samp-logger - Enregistreur structuré simple qui fournit un ensemble simple de fonctions pour la journalisation et le débogage.
  • TDW Recursion Scanner - Une bibliothèque qui trouve les récursions dans le code.
  • easyDialog - Facilite l'utilisation des boîtes de dialogue en général.
  • mdialog - Système de dialogue moderne, similaire à easyDialog.
  • Model Sizes Plus - Version mise à jour, avancée et plus précise des anciennes tailles de modèles incluses.
  • physics.inc - Simulez la physique 2D et 3D (mouvements réalistes, collisions, etc.).
  • samp-async-dialogs - Asyngestion chronométrée des dialogues avec les tâches PawnPlus.
  • speedcap.inc - Une bibliothèque pour contrôler la vitesse des véhicules.
  • SA:MP Command Guess - Un devineur de commande pour SA:MP utilisant la fonction Levenshtein Distance.
  • vending - Distributeurs automatiques côté serveur pour SA-MP.
  • strlib - Une liste de fonctions de chaîne utiles.
  • mathutil - Une liste de fonctions mathématiques utiles.
  • rotations.inc - Une liste de fonctions de rotation utiles par rotations.inc de Nero_3D.
  • ** [Fonctions de distance SA-MP] (https://github.com/Y-Less/samp-distance) ** - Une liste de fonctions de contrôle de distance utiles.
  • Nouveaux rappels SA-MP - Une liste de nouvelles fonctions de rappel utiles.
  • Dialogues alternatifs - Dialogues Textdraw avec un nouveau design.
  • eSelection - Ajoute la possibilité de créer des menus de sélection de modèles dynamiques dans vos modes de jeu SA-MP.
  • mSelection - Identique à eSelection, mais style différent.

Processeurs de commandes​

  • I-ZCMD - Version améliorée de ZCMD.
  • Pawn.CMD - Plus rapide que toute autre commande.
  • y_commands - Processeur de commandes d'YSI Inclut

Plugins​

  • JIT - Une fois que votre code est stable, utilisez ceci pour l'accélérer de manière significative.
  • CrashDetect - Outil de développement pour trouver des bugs lors des tests.
  • Profiler - Plugin de profileur de performances pour le serveur SA-MP.
  • sscanf - Convertir des chaînes en plusieurs valeurs, entiers, flottants, joueurs, etc.
  • MySQL Plugin - Connectez votre serveur à un serveur MySQL.
  • Streamer Plugin - Contourne de nombreuses limites SA:MP, comme les objets et les micros.
  • nativechecker - Vérification des natifs au démarrage du serveur.
  • FCNPC - Un plugin pour les serveurs SA-MP qui ajoute beaucoup de fonctionnalités aux PNJ standards existants.
  • FileManager - Vous permet de gérer des fichiers et des dossiers à partir du dossier racine (non limité au répertoire du serveur scriptfiles avec ce plugin).
  • Pawn.Raknet - Vous permet d'analyser le trafic RakNet.
  • samp-precise-timers - Un plugin pour SA-MP fournissant des minuteries précises, écrit en Rust.
  • PawnPlus - Étend les possibilités du langage de script Pawn avec de nouvelles constructions, types de données et techniques de programmation.
  • Accès à la mémoire PAWN - Permet d'allouer et de libérer de la mémoire du tas dans PAWN.
  • Native Fallback - Plugin SA-MP qui fournit une implémentation de secours aux natifs non enregistrés.
  • YSF - Un Plugin dont le but est de tirer le maximum de possibilités du serveur, principalement avec l'édition de la mémoire et le hooking.
  • SKY - Ce plugin fournit des fonctions de bas niveau pour permettre aux scripts Pawn de faire des choses puissantes.
  • Pawn.Regex - Ajoute la prise en charge des expressions régulières dans Pawn.
  • pawn-scraper - Un plugin de grattage puissant qui fournit une interface pour utiliser les html_parsers et les sélecteurs css dans le pion.
  • TOTP - Vous permet d'utiliser l'authentification TOTP dans un mode de jeu sa-mp.
  • Plug-in DNS - Ce plug-in fournit des fonctionnalités pour les recherches DNS et les recherches DNS inversées.
  • MapAndreas - Vous permet de charger différentes cartes de hauteur et de vérifier la hauteur minimale pour les coordonnées x,y.
  • ColAndreas - Crée une simulation du monde de San Andreas. Utilisation de la bibliothèque Bullet Physics.
  • PathFinder - Vous permet de calculer un itinéraire d'un point A à B sur la carte de San Andreas.
  • Custom Query Flood Check - Écrire des protections personnalisées contre la requête fbon.
  • sampml - Boîte à outils d'apprentissage automatique simplifiée et projets SAMP associés (Aimbot Detection).
  • TgConnector - Un plug-in de connecteur de télégramme qui permet d'interagir avec les robots de télégramme via SA-MP.
  • Connecteur Discord - Vous permet de contrôler un bot Discord depuis votre script PAWN.
  • TSConnector - Vous permet de contrôler un serveur Teamspeak3 depuis votre script PAWN.
  • Plugin IRC - Permet la création et la gestion de bots IRC via le serveur SA-MP.
  • pawn-requests - Fournit une API pour interagir avec les API HTTP(S) avec prise en charge des types de données texte et JSON.
  • pawn-redis - Accorder l'accès à une base de données rapide en mémoire et à des canaux de messages asynchrones entre d'autres programmes.
  • Chrono - Une bibliothèque Pawn moderne pour travailler avec les dates et les heures.
  • rustext - Correction du plugin de texte russe pour SA-MP : GameText, TextDraw et Menu.
  • Advanced SA NickName - Prise en charge de tous les symboles dans les surnoms.
  • SAMPSON - Un plugin JSON pour SA-MP.

Plugins client-serveur​

  • SA-MP+ - Une modification client qui utilise le plugin SDK de SA-MP pour interagir avec le serveur et ajouter de nouvelles fonctionnalités.
  • CHandling - Ajoute la possibilité d'attribuer des manipulations de véhicules à des véhicules individuels en modifiant le comportement par défaut du jeu.
  • SAMPVOICE - Implémentation de systèmes de communication vocale en langage Pawn pour les serveurs SA:MP.
  • KeyListener - Plugin client-serveur pour suivre l'appui sur n'importe quelle touche.
  • SAMP CEF - Plugins client et serveur pour SA:MP pour intégrer CEF.
  • SAMP_AC_v2 - Anti-triche côté client SA:MP.

Plugins cryptographiques​

  • whirlpool - Plugin cryptographique Whirlpool pour SA:MP.
  • bcrypt - Plugin cryptographique Bcrypt pour SA:MP.
  • samp-crypto - Une liste de fonctions cryptographiques telles que Argon2, Scrypt et Bcrypt pour SA:MP.
  • SHA512 - Plugin de chiffrement SHA512 pour SA:MP.

GDK/SDK​

  • sampsdk - C'est le strict minimum dont vous avez besoin pour développer des plugins.
  • sampgdk - Prise en charge du langage C/C++ pour écrire un mode de jeu pour SA:MP
  • SampSharp - Prise en charge du langage C# pour écrire un mode de jeu pour SA:MP
  • .NET Plugin - Prise en charge du langage C# pour écrire un plugin pour SA:MP
  • sampgo - Prise en charge de la langue Go pour écrire un mode de jeu/des plugins pour SA:MP
  • samp-node - Prise en charge du langage Javascript/Typescript pour écrire un mode de jeu pour SA:MP
  • Projet Shoebill - Prise en charge du langage Java pour écrire un mode de jeu pour SA:MP
  • pySAMP - Prise en charge du langage Python pour écrire un mode de jeu pour SA:MP
  • samp-rs - Prise en charge du langage Rust pour écrire un plugin pour SA:MP
  • Yet Another Lua Plugin - Prise en charge du langage Lua pour écrire un mode de jeu pour SA:MP
  • SAMPHP - Prise en charge du langage PHP pour écrire un mode de jeu pour SA:MP
  • SA-MP S[D]K - Prise en charge du langage D pour écrire un mode de jeu pour SA:MP
  • Kamp - Prise en charge du langage Kotlin pour l'écriture d'un mode de jeu pour SA:MP

Modes de jeu​

  • Example Gamemode - Listes des modes de jeu compatibles avec open.mp.
  • ScavengeSurvive - Mode de jeu basé sur la survie PvP.
  • gta-open - Un mode Flics et voleurs basé à Los Santos.
  • SF-CnR - Mode de jeu San Fierro Cops And Robbers pour SA-MP.
  • Jeu de rôle de nouvelle génération - Le mode de jeu Next Generation Gaming, LLC SA:MP.
  • SC-RP - Un mode de jeu de rôleavec la prise en charge de MySQL.

Filtrerscripts​

  • MapFix - Correction des bugs de texture de la carte GTA San Andreas.
  • SA:MP Animbrowse - Outil de navigation d'animation pour SA:MP.
  • IsPlayerUsingAndroid - Vérifie si un joueur est un utilisateur Android.
  • TextDraw-Editor - Un éditeur TextDraw doté de fonctionnalités riches.
  • Ultimate Creator - Éditeur de carte avancé avec de nombreuses fonctionnalités.
  • Éditeur de carte de Fusez - Un éditeur de carte en jeu pour SA:MP.
  • Texture Studio - Il s'agit d'un éditeur de cartes en jeu qui vous permettra de texturer jusqu'à l'indice de matériau 15 (16 matériaux) ainsi que de définir couleur pour cet index de matériau sur les objets.
