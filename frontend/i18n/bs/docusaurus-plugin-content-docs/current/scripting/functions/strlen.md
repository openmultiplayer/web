## Deskripcija

Dobij dužinu stringa.

| Ime            | Deskripcija              |
| -------------- | ------------------------ |
| const string[] | String za dobiti dužinu. |

## Returns

Dužina stringa kao cijeli broj.

## Primjeri

```c
new stringLength = strlen("Ovo je primjer stringa."); // stringLength je sata postavljenja na 24
```
