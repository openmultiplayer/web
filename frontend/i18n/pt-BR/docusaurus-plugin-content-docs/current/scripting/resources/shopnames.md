:::note

No GTA: SA (e também no SA-MP), certos scripts são pré-programados para rodar sempre que um jogador entra em determinado interior. Muitos interiores (principalmente lojas) têm seu próprio script externo. O problema é que há casos em que a capacidade de carregar e executar esses scripts externos sob demanda pode ser necessária. É aí que entra [SetPlayerShopName](../functions/SetPlayerShopName). Ela exige um nome de loja válido (ou uma string vazia para descarregar o script atual), que estão listados na tabela abaixo. Para informações adicionais sobre os interiores, consulte a [documentação de interiores](interiorids).

:::
