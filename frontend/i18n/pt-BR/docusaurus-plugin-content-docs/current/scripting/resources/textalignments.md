:::

| Value | Definition                        |
| ----- | --------------------------------- |
| 0     | OBJECT_MATERIAL_TEXT_ALIGN_LEFT   |
| 1     | OBJECT_MATERIAL_TEXT_ALIGN_CENTER |
| 2     | OBJECT_MATERIAL_TEXT_ALIGN_RIGHT  |
