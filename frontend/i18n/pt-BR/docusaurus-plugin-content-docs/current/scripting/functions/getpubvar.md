## Descrição

Obtém uma variável pública específica do script atual.

| Name         | Description                 |
| ------------ | --------------------------- |
| const name[] | O nome da variável pública. |

## Retorno

Naught caso a variável não exista (use [existpubvar](existpubvar) primeiro).

## Funções Relacionadas

- [setpubvar](setpubvar): Define uma variável pública específica no script atual.
- [existpubvar](existpubvar): Verifica se uma variável pública específica existe no script atual.
- [numpubvars](numpubvars): Conta quantas variáveis públicas existem no script.
