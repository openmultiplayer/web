## Descrição

Verifica se um menu está desabilitado.

| Name        | Description                   |
| ----------- | ----------------------------- |
| Menu:menuid | O ID do menu a ser verificado. |

## Retorno

Retorna **true** se o menu estiver desabilitado, caso contrário **false**.

## Funções Relacionadas

- [IsValidMenu](IsValidMenu): Verifica se um ID de menu é válido.
- [IsMenuRowDisabled](IsMenuRowDisabled): Verifica se uma linha do menu está desabilitada.
