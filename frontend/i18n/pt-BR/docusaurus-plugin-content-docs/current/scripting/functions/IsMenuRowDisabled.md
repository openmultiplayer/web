## Descrição

Verifica se uma linha de menu está desabilitada.

| Name        | Description                   |
| ----------- | ----------------------------- |
| Menu:menuid | O ID do menu a ser verificado. |

## Retorno

Retorna **true** se a linha estiver desabilitada, caso contrário **false**.

## Funções Relacionadas

- [IsValidMenu](IsValidMenu): Verifica se um ID de menu é válido.
- [IsMenuDisabled](IsMenuDisabled): Verifica se um menu está desabilitado.
