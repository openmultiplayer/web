## Descrição

Obtém o ângulo do reator do hydra do veículo do jogador.

## Parâmetros

| Name     | Description          |
| -------- | -------------------- |
| playerid | O ID do jogador.     |

## Exemplos

```c
new reactorAngle = GetPlayerHydraReactorAngle(playerid);
```
