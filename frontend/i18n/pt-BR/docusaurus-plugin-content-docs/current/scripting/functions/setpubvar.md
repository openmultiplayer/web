## Descrição

Define uma variável pública específica no script atual.

| Name         | Description                                    |
| ------------ | ---------------------------------------------- |
| const name[] | O nome da variável pública.                    |
| value        | O valor a ser armazenado na variável pública.  |

## Retorno

O valor anterior da variável.

## Funções Relacionadas

- [getpubvar](getpubvar): Obtém uma variável pública específica do script atual.
- [existpubvar](existpubvar): Verifica se uma variável pública específica existe no script atual.
- [numpubvars](numpubvars): Conta quantas variáveis públicas existem no script.
