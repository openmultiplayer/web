## Descrição

Obtém o clima atual do mundo.

## Exemplos

```c
SetWeather(19);

printf("Clima atual do mundo: %d", GetWeather());
// A saída será 'Clima atual do mundo: 19'
```

## Funções Relacionadas

- [SetWeather](SetWeather): Define o clima do mundo para todos os jogadores.
- [GetPlayerWeather](GetPlayerWeather): Obtém o clima de um jogador.
- [SetPlayerWeather](SetPlayerWeather): Define o clima de um jogador.
