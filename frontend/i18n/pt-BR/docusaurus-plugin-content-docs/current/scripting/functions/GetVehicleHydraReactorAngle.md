## Descrição

Obtém o ângulo do reator do hydra do veículo.

## Parâmetros

| Name      | Description           |
| --------- | --------------------- |
| vehicleid | O ID do veículo.      |

## Exemplos

```c
new reactorAngle = GetVehicleHydraReactorAngle(vehicleid);
```
