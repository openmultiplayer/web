## Descrição

Verifica se um ID de modelo customizado é válido.

| Name    | Description                                |
| ------- | ------------------------------------------ |
| modelid | O ID do modelo customizado a ser verificado. |

## Retorno

**true** - O ID de modelo customizado é válido.

**false** - O ID de modelo customizado não é válido.
