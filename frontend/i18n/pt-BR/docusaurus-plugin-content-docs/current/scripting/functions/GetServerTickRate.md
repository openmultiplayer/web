## Descrição

Obtém a taxa de ticks (como FPS) do servidor.

## Exemplos

```c
printf("A taxa de ticks atual do servidor é: %i", GetServerTickRate());
```

## Funções Relacionadas

- [GetNetworkStats](GetNetworkStats): Obtém as estatísticas de rede do servidor e salva em uma string.
