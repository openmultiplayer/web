## Descrição

Esta função retorna o valor absoluto de um float.

| Name        | Description                                   |
| ----------- | --------------------------------------------- |
| Float:value | O valor float para obter o valor absoluto.    |

## Retorno

O valor absoluto do float (como valor float).

## Exemplos

```c
new Float:value;

value = floatabs(47.0); // Isso retornará 47.0

value = floatabs(-47.0); // Isso retornará o mesmo.
```
