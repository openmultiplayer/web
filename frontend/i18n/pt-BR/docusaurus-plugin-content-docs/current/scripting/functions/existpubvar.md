## Descrição

Verifica se uma variável pública específica existe no script atual.

| Name         | Description                          |
| ------------ | ------------------------------------ |
| const name[] | O nome da variável pública.          |

## Retorno

A variável existe? (true/false)

## Funções Relacionadas

- [getpubvar](getpubvar): Obtém uma variável pública específica do script atual.
- [setpubvar](setpubvar): Define uma variável pública específica no script atual.
- [numpubvars](numpubvars): Conta quantas variáveis públicas existem no script.
