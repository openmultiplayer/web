## Description

Gets the hydra reactor angle of the vehicle.

## Parameters

| Name      | Description            |
|-----------|------------------------|
| vehicleid | The ID of the vehicle. |

## Examples

```c
new reactorAngle = GetVehicleHydraReactorAngle(vehicleid);
```
