## Description

Set the maximum distance to display the names of players.

| Name           | Description          |
| -------------- | -------------------- |
| Float:distance | The distance to set. |

## Returns

This function does not return any specific values.

## Examples

```c
SetNameTagDrawDistance(20.0);
```

## Notes

:::tip

Default distance is 70.0 SA units

:::
