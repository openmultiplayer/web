---

:::note

这些文件模式适用于[fopen](../functions/fopen)函数。

:::

| 模式         | 说明                                             |
| ------------ | ------------------------------------------------ |
| io_read      | 读取文件内容                                     |
| io_write     | 写入文件（不存在则创建），会清空所有现有内容     |
| io_readwrite | 读写文件（不存在则创建）                         |
| io_append    | 追加写入文件（仅写入模式），文件不存在时自动创建 |
