## Description

Get the length of a string.

| Name           | Description                      |
| -------------- | -------------------------------- |
| const string[] | The string to get the length of. |

## Returns

The length of the string as an integer.

## Examples

```c
new stringLength = strlen("This is an example string."); // stringLength is now set to 26
```
