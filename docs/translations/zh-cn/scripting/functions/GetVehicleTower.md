## Description

Get the ID of the cab attached to a vehicle.

| Name      | Description                                  |
| --------- | -------------------------------------------- |
| vehicleid | The ID of the vehicle to get the cab of.     |

## Returns

The vehicle ID of the cab or **0** if no cab is attached.

## Examples

```c
new cabId = GetVehicleTower(vehicleid);
```
