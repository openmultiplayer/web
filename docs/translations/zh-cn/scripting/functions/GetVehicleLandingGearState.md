## Description

Gets the current vehicle [landing gear state](../resources/landinggearstate) from the latest driver.

## Parameters

| Name      | Description            |
|-----------|------------------------|
| vehicleid | The ID of the vehicle. |

## Examples

```c
new LANDING_GEAR_STATE:state = GetVehicleLandingGearState(vehicleid);
```
