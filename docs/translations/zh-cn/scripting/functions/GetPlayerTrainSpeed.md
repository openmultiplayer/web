## Description

Gets the speed of the player's train.

## Parameters

| Name     | Description           |
|----------|-----------------------|
| playerid | The ID of the player. |

## Examples

```c
new Float:speed = GetPlayerTrainSpeed(playerid);
SendClientMessage(playerid, 0xFFFF00FF, "The speed of your train: %f", speed);
```
