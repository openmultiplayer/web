## Description

Checks if a vehicle siren is on or off.

## Parameters

| Name      | Description                     |
|-----------|---------------------------------|
| vehicleid | The ID of the vehicle to check. |

## Return Values

**true** - Vehicle siren is on.

**false** - Vehicle siren is off.

## Examples

```c
if (IsVehicleSirenEnabled(vehicleid))
{
    // Vehicle siren is on
}
else
{
    // Vehicle siren is off
}
```
