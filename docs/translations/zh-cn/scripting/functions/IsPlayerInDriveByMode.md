## Description

Check if the player is in driveby mode.

| Name     | Description                                                 |
| -------- | ----------------------------------------------------------- |
| playerid | The ID of the player to check.       |

## Returns

Returns true if the player is in driveby mode, otherwise false.

## Examples

```c
if (IsPlayerInDriveByMode(playerid))
{
    // do something
}
```
