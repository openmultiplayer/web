## Description

Get the player animation flags.

| Name     | Description                              |
| -------- | ---------------------------------------- |
| playerid | The player id you want to get the animation flags from |

## Returns

Returns the player animation flags as integer.

## Examples

```c
new flags = GetPlayerAnimFlags(playerid);
```
