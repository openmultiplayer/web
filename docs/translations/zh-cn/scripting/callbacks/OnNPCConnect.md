## 描述

当 NPC 成功连接到服务器时会调用该回调。

| 参数名     | 描述                     |
| ---------- | ------------------------ |
| myplayerid | 已经赋予该 NPC 的玩家 ID |

## 案例

```c
public OnNPCConnect(myplayerid)
{
    printf("我成功连接了服务器, ID %i!", myplayerid);
}
```

- [OnNPCDisconnect](../callbacks/OnNPCDisconnect): 当 NPC 与服务器断开连接时调用。
