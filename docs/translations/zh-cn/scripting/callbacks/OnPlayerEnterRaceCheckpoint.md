## 描述

当玩家进入一个比赛检查点时，这个回调函数被调用。

| 参数名   | 描述                        |
| -------- | --------------------------- |
| playerid | 进入比赛检查点的玩家的 ID。 |

## 返回值

它在过滤脚本中总是先被调用。

## 案例

```c
public OnPlayerEnterRaceCheckpoint(playerid)
{
    printf("玩家 %d 进入了一个比赛检查点!", playerid);
    return 1;
}
```
