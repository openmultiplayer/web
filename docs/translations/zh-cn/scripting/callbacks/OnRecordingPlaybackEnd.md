## 描述

当使用 NPC:StartRecordingPlayback 复制的录制文件到达其末尾时，将调用此回调。

## 案例

```c
public OnRecordingPlaybackEnd()
{
    StartRecordingPlayback(PLAYER_RECORDING_TYPE_DRIVER, "all_around_lv_bus"); //这将在录制的文件完成复制后再次启动。
}
```
