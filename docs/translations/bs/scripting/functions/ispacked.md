## Deskripcija

Provjerava je li zadati string spakovan.

| Ime    | Deskripcija           |
| ------ | --------------------- |
| string | String za provjeriti. |

## Returns

1 ako je string upakovan, 0 if ako je raspakiran.

## Primjeri

```c
// Kreiraj upakovani string
new string[24 char];

if (ispacked(string))
{
   print("String je spakovan.");
}
```
