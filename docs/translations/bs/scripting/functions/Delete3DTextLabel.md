## Deskripcija

Obriši 3D text label (kreiran sa Create3DTextLabel).

| Ime       | Deskripcija                    |
| --------- | ------------------------------ |
| Text3D:textid | ID 3D text labela za obrisati. |

## Returns

1 ako je 3D text label obrisan, uostalom 0.

## Primjeri

```c
new Text3D:gMyLabel;

gMyLabel = Create3DTextLabel(...);

Delete3DTextLabel(gMyLabel);
```
