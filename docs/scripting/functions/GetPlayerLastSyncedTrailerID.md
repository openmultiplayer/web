## Description

Gets the player's last synced trailer ID.

## Parameters

| Name     | Description           |
|----------|-----------------------|
| playerid | The ID of the player. |

## Return Values

Returns the last synced trailer ID.

## Examples

```c
new trailerid = GetPlayerLastSyncedTrailerID(playerid);
```
