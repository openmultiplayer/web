## Description

Can the player use weapons?

| Name     | Description                    |
| -------- | ------------------------------ |
| playerid | The ID of the player to check. |

## Returns

**true** - Player is allowed.

**false** - Player is not allowed.

## Examples

```c
public OnPlayerSpawn(playerid)
{
    if (ArePlayerWeaponsAllowed(playerid))
    {
        // do something
    }

    return 1;
}
```
