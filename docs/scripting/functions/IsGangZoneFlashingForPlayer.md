## Description

Check if the gangzone is flashing for player.

| Name        | Description                               |
| ----------- | ----------------------------------------- |
| playerid    | The ID of the player to be checked.       |
| zoneid      | The ID of the gangzone.                   |

## Returns

1: The gangzone is flashing for player.

0: The gangzone is not flashing for player.
