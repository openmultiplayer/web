## Description

Check if the player camera target is enabled.

| Name     | Description          |
| -------- | -------------------- |
| playerid | The ID of the player |

## Returns

**true** - Enable.

**false** - Disable.

## Examples

```c
public OnPlayerSpawn(playerid)
{
    if (IsPlayerCameraTargetEnabled(playerid))
    {
        // do something
    }

    return 1;
}
```
