## Description

Set the NPC's facing angle.

| Name                 | Description                   |
| -------------------- | ----------------------------- |
| Float:ang            | The new NPC's facing angle.   |

## Examples

```c
SetMyFacingAngle(0); //NPC faces north
```

```c
       north (0)
          |
(90) west- -east (270)
          |
       south (180)
```
