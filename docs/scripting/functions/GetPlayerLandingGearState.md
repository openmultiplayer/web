## Description

Gets the [landing gear state](../resources/landinggearstate) of the current player's vehicle.

## Parameters

| Name     | Description           |
|----------|-----------------------|
| playerid | The ID of the player. |

## Examples

```c
new LANDING_GEAR_STATE:state = GetPlayerLandingGearState(playerid);
```
