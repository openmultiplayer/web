## Description

Check if a vehicle is occupied.

## Parameters

| Name      | Description            |
|-----------|------------------------|
| vehicleid | The ID of the vehicle. |

## Return Values

**true** - Vehicle is occupied.

**false** - Vehicle is not occupied.

## Examples

```c
if (IsVehicleOccupied(vehicleid))
{
    // Vehicle is occupied
}
else
{
    // Vehicle is not occupied
}
```
