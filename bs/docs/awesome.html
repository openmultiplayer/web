Awesome

Alati​

  • Community Compiler - Izrazito ažurirana verzija kompajlera, s mnogim popravcima i poboljšanjima.
  • sampctl - Upravitelj paketa za instaliranje libraries-a i pokretanje vašeg servera.
  • Plugin Runner - Alat za pokretanje lagane verzije poslužitelja izravno s naredbenog retka (nije potreban server.cfg) za testiranje plugina.
  • Plugin Boilerplate - Pravljenje plugina lakše sa boilerplate-om.
  • SA:MP Plugin Template Library - Izrada vlastitih plugina vrlo jednostavna i brza sa ovom bibliotekom šablona.
  • SA-MP Fiddle - Mjesto za testiranje vaših skripti (isječci, PoC, otklanjanje grešaka, itd.) i dijeljenje s drugima.
  • Pawn Syntax - Sublime - Pawn se samodovršava za Sublime Text.
  • Pawn Syntax - Visual Marketplace - Pawn se samodovršava za Visual Studio Code.
  • SA-MP Zone Editor - Zone Editor za izradu područja i ostalog.
  • SA-MP Map Editor - Popularni Map Editor za SA:MP.

Biblioteke​

  • SA:MP stdlib - Ažurirane verzije zadanih include-ova. const-ispravno, dokumentovano, i dovršeno.
  • fixes.inc - Izuzetno optimizirani popravci za veliki broj programskih pogrešaka SA:MP. Uključi i igraj.
  • YSI-Includes - Najstariji, najveći i najviše testirani, dobro podržani library za SA:MP/pawn koji nudi ogroman broj novih značajki igre i jezika.
  • foreach - foreach samostalni include (verzija koja nije y_iterate).
  • amx_assembly - Pristup niske razine samoj založnoj skripti.
  • md-sort - Sortiraj višedimenzionalne nizove u Pawn-u. Modificira interne pokazivače niza umjesto kopiranja podataka.
  • indirection - Sustav za neizravno prosljeđivanje pokazivača i pozivanje funkcija, sa apstraktnim metodama za prilagodbu poziva.
  • code-parse.inc - Analizirajte i prilagodite založni kod u vrijeme kompajlanja.
  • Extended Vehicle Information - Velike količine zastavica vozila iz jednog igrača u SQLite bazu podataka.
  • sqlitei - API visoke razine za SA-MP SQLite funkcije.
  • weapon-config - Dosljedniji i osjetljiviji sistem oštećenja s mnogo novih funkcija.
  • geo-ip.inc - Geografska informacija bazirana na vašoj IP Adresi.
  • progress2.inc - Prikaži vrijeme, health, gorivo vozila i ostale stvari kao progresivan bar.
  • weapon-data.inc - Fino podešeni, prilagodljivi podaci o oružju.
  • MV_Youtube.inc - Youtube Konverter API za Audio Stream.
  • MySQL Prepared Statements - Emuliranje pripremljene izjave za PAWN MySQL plugin.
  • samp-server-weapons - Oružje koje u potpunosti kontroliše server.
  • actor_robbery - Inspirisan pljačkom prodavnice GTA V. actor_robbery.inc to imitira!
  • samp-aviation - Osnovni autopilot zasnovan na visini i nagibu za SA-MP.
  • samp-logger - Jednostavan strukturirani loger koji pruža jednostavan skup funkcija za evidentiranje i otklanjanje grešaka.
  • TDW Recursion Scanner - Biblioteka koja pronalazi rekurzije u kodu.
  • easyDialog - Olakša korištenje dijaloga općenito.
  • mdialog - Moderni dijalog sistem, sličan easyDialog-u.
  • Model Sizes Plus - Ažurirana, napredna, preciznija verzija starog modelsizes include-a.
  • physics.inc - Simuliraj 2D i 3D fiziku (realistični pokreti, kolizije i još mnogo toga).
  • samp-async-dialogs - Asinkrono rukovanje dijalogom sa PawnPlus zadacima.
  • speedcap.inc - Include za kontrolu brzine vozila.
  • SA:MP Command Guess - Pogađač komandi za SA:MP koristeći Levenshtein Distance funkciju.
  • vending - Server-side automati za SA-MP.
  • strlib - Lista korisnih string funkcija.
  • mathutil - Lista korisnih matematičkih funkcija.
  • rotations.inc - Lista korisnih funkcija rotacije iz Nero_3D-ov rotations.inc.
  • SA-MP Distance Functions - Lista korisnih funkcija provjere udaljenosti.
  • New SA-MP callbacks - Lista korisnih novih callback funkcija.
  • Alternative Dialogs - Textdraw dijalozi s novim dizajnom.
  • eSelection - Dodaje mogućnost kreiranja dinamičkih izbornika za odabir modela u vašim SA-MP gamemode-ovima.
  • mSelection - Isto kao eSelection, ali drugačiji stil.
  • RGB-Textdraw - Dodaje RGB u textdraw.

Komandni Procesori​

  • I-ZCMD - Poboljšana verzija ZCMD-a.
  • Pawn.CMD - Brži od bilo kojeg drugog komandnog procesora.
  • y_commands - Komandni procesori iz YSI Include-ova

Plugini​

  • JIT - Nakon što je vaš kod stabilan, upotrijebite ga da biste ga znatno ubrzali.
  • CrashDetect - Razvojni alat za pronalaženje bugova tijekom testiranja.
  • Profiler - Plugin za profilisanje performansi za SA-MP server.
  • sscanf - Pretvori nizove u više vrijednosti, intove, plutajuće, playere itd.
  • MySQL Plugin - Konektujte svoj server sa MySQL serverom.
  • Streamer Plugin - Zaobiđite mnoga SA:MP ograničenja, poput predmeta i preuzimanja.
  • nativechecker - Provjerava native-e kad se server pokrene.
  • FCNPC - Plugin za SA-MP servere koji dodaje puno mogućnosti postojećim standardnim NPC-ovima. Trenutno ne radi u open.mp, od januara 2023.
  • FileManager - Omogućava vam da upravljate datotekama i fasciklama iz osnovnog foldera (nije ograničeno na direktorijum servera scriptfiles-a sa ovim pluginom).
  • Pawn.Raknet - Omogućava vam da analizirate RakNet promet.
  • samp-precise-timers - Plugin za SA-MP koji pruža precizne tajmere, napisan u Rustu. Nije potreban u open.mp, jer open.mp ima precizne tajmere.
  • PawnPlus - Proširuje mogućnosti skriptnog jezika Pawn novim konstrukcijama, tipovima podataka i tehnikama programiranja.
  • PAWN memory access - Omogućava dodjelu i oslobađanje memorije iz gomile u PAWN-u.
  • Native Fallback - SA-MP plugin koji pruža zamjensku implementaciju neregistriranim urođenicima.
  • YSF - Plugin čija je svrha da izvuče maksimalne mogućnosti sa servera, uglavnom sa uređivanjem memorije i spajanjem. Mnoge njegove funkcije su sada već u open.mp. Pogledajte ovaj github problem za listu funkcija.
  • SKY - Ovaj plugin pruža funkcije niskog nivoa koje omogućavaju Pawn skriptama da rade moćne stvari.
  • Pawn.Regex - Dodaje podršku za regularne izraze u Pawn.
  • pawn-scraper - Moćan plugin za struganje koji pruža interfejs za korišćenje html_parsera i css selektora u zalogu.
  • TOTP - Omogućava vam da koristite TOTP autentifikaciju u SA-MP gamemode-u.
  • DNS Plugin - Ovaj plugin pruža funkcionalnost za DNS traženje i obrnuto DNS traženje.
  • MapAndreas - Omogućava vam da učitate različite visinske karte i provjerite minimalnu visinu za x, y koordinate.
  • ColAndreas - Kreira simulaciju svijeta San Andreas-a. Korištenje biblioteke Bullet Physics.
  • PathFinder - Omogućuje vam izračunavanje rute od tačke A do B na mapi San Andreas-a.
  • Custom Query Flood Check - Napišite prilagođene zaštite od query flood-ova.
  • sampml - Pojednostavljeni alati za mašinsko učenje i povezani SAMP projekti (Detekcija Aimbot-a).
  • TgConnector - Plugin za telegram konektor koji pomaže u interakciji s telgram botovima putem SA-MP-a.
  • Discord Connector - Omogućava vam da kontrolišete Discord bot iz vaše PAWN skripte.
  • TSConnector - Omogućava vam da kontrolišete Teamspeak3 server iz vaše PAWN skripte.
  • IRC Plugin - Omogućava kreiranje i upravljanje IRC botovima preko SA-MP servera.
  • pawn-requests - Pruža API za interakciju s HTTP(S) API-jima s podrškom za tekstualne i JSON tipove podataka.
  • pawn-redis - Omogućavanje pristupa brzoj bazi podataka u memoriji i asinhronim kanalima poruka između drugih programa.
  • Chrono - Moderna Pawn biblioteka za rad sa datumima i vremenima.
  • rustext - Popravljen ruski tekst plugin za SA-MP: GameText-ove, TextDraw-ove i Menije.
  • Advanced SA NickName - Podrška za sve simbole u nadimcima.
  • SAMPSON - JSON plugin za SA-MP.

Klijent-Server Plugini​

  • SA-MP+ - Modifikacija klijenta koja koristi SA-MP plugin SDK za interakciju sa serverom i dodavanje novih funkcija.
  • CHandling - Dodaje mogućnost dodjeljivanja upravljanja vozilom pojedinačnim vozilima mijenjanjem zadanog ponašanja igre.
  • SAMPVOICE - Implementacija sistema glasovne komunikacije na jeziku Pawn za SA:MP servere.
  • KeyListener - Klijent-server plugin za praćenje pritiska bilo koje tipke.
  • SAMP CEF - Klijentski i serverski plugin za SA:MP za ugradnju CEF-a.
  • SAMP_AC_v2 - SA:MP Anti-Cheat integrisan u klijent.

Crypto Plugini​

  • whirlpool - Whirlpool kriptografski plugin za SA:MP.
  • bcrypt - Bcrypt kriptografski plugin za SA:MP.
  • samp-crypto - Lista kriptografskih funkcija kao što su Argon2, Scrypt i Bcrypt za SA:MP.
  • SHA512 - SHA512 kriptografski plugin za SA:MP.

GDK/SDKs​

Sada možete napraviti svoje skripte na drugim jezicima osim pawn-a bez ikakvog plugina koristeći open.mp. Pogledajte ovaj blog post

  • sampsdk - Ovo je minimum koji vam je potreban za razvoj dodataka.
  • sampgdk - Podrška jezika C/C++ za pisanje gamemode-a za SA:MP.
  • SampSharp - Podrška jezika C# za pisanje gamemode-a za SA:MP.
  • .NET Plugin - Podrška jezika C# za pisanje plugina za SA:MP.
  • sampgo - Go jezična podrška za pisanje gamemode-a/plugina za SA:MP.
  • samp-node - Javascript/Typescript jezička podrška za pisanje gamemode-a za SA:MP.
  • Shoebill Project - Podrška za Java jezik za pisanje gamemode-a za SA:MP.
  • pySAMP - Podrška jezika Python za pisanje gamemode-a za SA:MP.
  • samp-rs - Podrška za Rust jezik za pisanje plugina za SA:MP.
  • Yet Another Lua Plugin - Podrška za Lua jezik za pisanje gamemode-a za SA:MP.
  • SAMPHP - PHP jezička podrška za pisanje gamemode-a za SA:MP.
  • SA-MP S[D]K - D jezička podrška za pisanje gamemode-a za SA:MP.
  • Kamp - Kotlin jezička podrška za pisanje gamemode-a za SA:MP.

Filterskripte​
