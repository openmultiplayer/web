OnNPCModeExit

Description​

This callback is called when a NPC script unloaded.

Examples​

public OnNPCModeExit()
{
    print("NPC script unloaded");
    return 1;
}

Related Callbacks​

The following callbacks might be useful, as they're related to this callback in one way or another.
