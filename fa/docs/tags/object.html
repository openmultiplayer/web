BeginObjectEditing

به بازیکن اجازه می‌دهد تا یک شی‌ء (موقعیت و چرخش) را با استفاده از ماوس در GUI (رابط گرافیکی کاربر) ویرایش کند.

BeginObjectSelecting

مکان‌نما را نمایش می‌دهد و به بازیکن اجازه انتخاب یک شی‌ء را می‌دهد.

BeginPlayerObjectEditing

به بازیکنان اجازه می‌دهد تا یک شی‌ء بازیکن (موقعیت و چرخش) را با GUI و ماوس خود ویرایش کنند.

CreateObject

CreatePlayerObject

شیئی ایجاد می‌کند که فقط برای یک بازیکن قابل مشاهده است.

DestroyObject

آبجکتی که با CreateObject ایجاد شده را نابود (حذف) می‌کند.

DestroyPlayerObject

آبجکت بازیکن ایجاد شده با CreatePlayerObject را نابود می‌کند.

EditAttachedObject

ورود به حالت ویرایش برای آبجکت چسبانده شده.

EndObjectEditing

GetObjectModel

GetObjectMoveSpeed

GetObjectMovingTargetRot

GetObjectPos

GetObjectRot

چرخش فعلی یه آبجکت رو برمیگردونه.

GetObjectSyncRotation

sync rotation یک آبجکت رو می‌گیره.

GetObjectType

نوع یک آبجکت رو می‌گیره. (global یا player)

GetPlayerAttachedObject

GetPlayerCameraTargetPlayerObject

GetPlayerObjectAttachedData

دریافت داده‌های اتصال یک player-object.

GetPlayerObjectAttachedOffset

GetPlayerObjectMovingTargetPos

GetPlayerObjectMovingTargetRot

GetPlayerObjectSyncRotation

HasPlayerObjectCameraCollision

IsPlayerAttachedObjectSlotUsed

IsPlayerObjectMaterialSlotUsed

IsValidObject

IsValidPlayerObject

MoveObject

OnObjectMoved

این کالبک پس از حرکت یک شی با MoveObject فراخوانده می‌شود (زمانی که حرکت متوقف می‌شود).

OnPlayerEditAttachedObject

این کالبک زمانی فراخوانده می‌شود که بازیکن حالت ویرایش شیء متصل شده را پایان می‌دهد.

OnPlayerEditObject

این کالبک زمانی فراخوانده می‌شود که یک بازیکن ویرایش یک آبجکت را تمام کند (BeginObjectEditing/BeginPlayerObjectEditing).

OnPlayerObjectMoved

این کالبک زمانی فراخوانده می‌شود که یک player object پس از MovePlayerObject حرکت کند (زمانی که متوقف می‌شود).

OnPlayerSelectObject

این کالبک زمانی فراخوانده می‌شود که بازیکن آبجکتی را پس از استفاده از BeginObjectSelecting انتخاب کند.

SetObjectMoveSpeed

SetObjectPos

SetObjectRot

SetObjectsDefaultCameraCol

اجازه می‌ده که برخورد دوربین با object‌های تازه ساخته شده به طور پیش‌فرض غیرفعال بشه.

SetObjectsDefaultCameraCollision

اجازه می‌ده که برخورد دوربین با object‌های تازه ساخته شده به طور پیش‌فرض غیرفعال بشه.

SetPlayerAttachedObject

SetPlayerObjectMaterial

SetPlayerObjectMaterialText

texture یک player object رو با متن جایگزین می‌کنه.

SetPlayerObjectMoveSpeed

سرعت حرکت یک player object رو تنظیم می‌کنه.

SetPlayerObjectPos

موقعیت یک player-object رو به مختصات مشخص شده تنظیم می‌کنه.

SetPlayerObjectRot
